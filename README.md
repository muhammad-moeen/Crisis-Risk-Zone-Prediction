## ⚡ What is CRZP?

**CRZP** is a high-performance, cloud-native intelligence engine designed to process dynamic global hazards. It fuses continuous real-time incident data with an advanced Machine Learning ensemble to output a precise, composite **Risk Severity Index (RSI)**. 

The system bypasses delayed reports by computing live threat scores on demand, backed by automated language briefings that explain exactly *why* a particular location carries its risk rating.
